# 前端手工 QA 清单 - 购物车、结算与订单

**特性**: 003-cart-checkout-orders
**版本**: 1.0
**创建日期**: 2025-11-29
**目的**: 确保购物车、结算、优惠券、支付和订单管理功能在真实环境中正常工作

## 测试环境准备

- [ ] 开发服务器运行正常（`npm run dev`）
- [ ] 后端 API 服务可访问（通常是 localhost:8080）
- [ ] 已有测试账号并能正常登录
- [ ] 数据库中有可用的商品数据
- [ ] 数据库中有可用的优惠券数据

## 1. 购物车基本功能（US1）

### 1.1 添加商品到购物车
- [ ] 从商品列表页点击"加入购物车"，商品成功添加
- [ ] 购物车图标显示正确的商品数量
- [ ] 进入购物车页面，能看到刚添加的商品

### 1.2 数量调整
- [ ] 点击"+"按钮，数量增加，小计和总计正确更新
- [ ] 点击"-"按钮，数量减少，小计和总计正确更新
- [ ] 直接输入数字（如 5），数量更新正确
- [ ] 输入 0 或负数，数量回退到上次有效值并显示提示
- [ ] 输入 100 或更大数字，数量回退到 99 并显示提示
- [ ] 输入非数字字符（如 abc），数量回退并显示提示

### 1.3 删除商品
- [ ] 点击删除按钮，显示确认弹窗
- [ ] 确认删除后，商品从列表移除
- [ ] 总计金额正确更新
- [ ] 取消删除，商品保留在列表中

### 1.4 空购物车状态
- [ ] 删除所有商品后，显示空态提示
- [ ] 空态包含"返回菜单"或类似 CTA 按钮
- [ ] 点击 CTA 能正确跳转到商品列表

### 1.5 购物车持久化
- [ ] 添加商品后刷新页面，购物车内容保持
- [ ] 退出登录后重新登录，购物车内容从服务器恢复

## 2. 购物车到结算流程（US2）

### 2.1 进入结算页
- [ ] 购物车有商品时，"去结算"按钮可点击
- [ ] 点击"去结算"，跳转到结算页面
- [ ] 结算页显示购物车商品快照（名称、数量、单价、小计）
- [ ] 商品快照与购物车内容一致

### 2.2 结算表单填写
- [ ] 可以选择"自提"或"配送"
- [ ] 选择自提时，显示姓名和电话输入框
- [ ] 选择配送时，显示地址选择/输入框
- [ ] 必填字段未填写时，"支付"按钮禁用或显示提示
- [ ] 备注输入框限制 120 字符，超出时显示提示

### 2.3 预览触发
- [ ] 进入结算页时自动触发预览
- [ ] 切换自提/配送时触发预览
- [ ] 选择/移除优惠券时触发预览
- [ ] 预览加载时显示加载状态
- [ ] 预览成功后显示价格拆分和光阴值预估

### 2.4 返回购物车
- [ ] 点击返回按钮，回到购物车页面
- [ ] 已填写的表单内容保留（再次进入结算时恢复）

## 3. Buy Now 独立流程（US3）

### 3.1 Buy Now 入口
- [ ] 商品详情页有"立即购买"按钮
- [ ] 点击"立即购买"，直接进入结算页
- [ ] 结算页显示该商品的快照

### 3.2 购物车隔离
- [ ] Buy Now 结算时，购物车内容不变
- [ ] 完成 Buy Now 支付后，查看购物车，原内容仍在
- [ ] Buy Now 中途返回，购物车内容不受影响

### 3.3 Buy Now 中断恢复
- [ ] Buy Now 进入结算后，点击浏览器后退
- [ ] 再次进入结算，Buy Now 快照仍可继续

## 4. 优惠券选择与自动失效（US4）

### 4.1 优惠券列表
- [ ] 结算页显示"选择优惠券"入口
- [ ] 点击后显示可用和不可用优惠券分组
- [ ] 可用优惠券显示名称、折扣、门槛、有效期
- [ ] 不可用优惠券显示具体原因（如"未达到使用门槛"）

### 4.2 选择优惠券
- [ ] 选择一张可用优惠券，立即触发预览
- [ ] 价格拆分中显示折扣金额
- [ ] 应付金额 = 商品总额 - 折扣
- [ ] 光阴值预估根据应付金额更新

### 4.3 优惠券自动移除
- [ ] 选择优惠券后，减少购物车商品数量
- [ ] 如果金额低于优惠券门槛，优惠券自动移除
- [ ] 显示 toast 或 inline 提示说明移除原因
- [ ] 价格拆分恢复为无折扣状态

### 4.4 切换优惠券
- [ ] 选择优惠券 A 后，再选择优惠券 B
- [ ] 优惠券 A 自动取消，优惠券 B 生效
- [ ] 价格拆分正确更新

## 5. 价格拆分与模拟支付（US5）

### 5.1 价格拆分显示
- [ ] 结算页始终显示价格拆分组件
- [ ] 显示：商品总额、优惠金额、应付金额（高亮）
- [ ] 显示：预计获得的光阴值
- [ ] 移动端布局不溢出，可折叠/展开

### 5.2 支付提交
- [ ] 填写完整表单后，"支付"按钮可点击
- [ ] 点击"支付"，显示加载状态
- [ ] 支付成功后，跳转到订单详情页

### 5.4 预览过期处理

- [ ] 点击"支付"时，提示预览已过期
- [ ] 自动刷新预览，显示最新价格
- [ ] 用户确认后才能继续支付

### 5.5 价格/库存变动
- [ ] 结算过程中，后端修改商品价格或库存
- [ ] 点击"支付"时，显示友好错误提示
- [ ] 提示用户刷新购物车或重新预览
- [ ] 已填写的表单内容不丢失

## 6. 订单列表（US6）

### 6.1 订单列表加载
- [ ] 进入订单列表页，显示加载状态
- [ ] 加载完成后，显示订单卡片列表
- [ ] 每个卡片显示：订单号、状态、时间、商品预览、金额

### 6.2 状态筛选
- [ ] 顶部显示状态标签：全部、待支付、制作中、已完成、已取消、退款中、已退款
- [ ] 点击不同标签，列表内容正确切换
- [ ] 标签显示对应状态的订单数量
- [ ] 切换标签时，已加载的列表从缓存恢复（不重新请求）

### 6.3 分页/加载更多
- [ ] 订单较多时，显示"加载更多"按钮
- [ ] 点击"加载更多"，追加新订单到列表末尾

### 6.4 空态和错误态
- [ ] 某个状态下无订单时，显示空态提示
- [ ] 网络错误时，显示错误提示和"重试"按钮
- [ ] 点击"重试"，重新加载列表

### 6.5 快捷动作
- [ ] 待支付订单卡片显示"继续支付"和"查看详情"按钮
- [ ] 其他状态订单只显示"查看详情"按钮
- [ ] 点击"继续支付"，跳转到支付确认页
- [ ] 点击"查看详情"，跳转到订单详情页

## 8. 取消待支付订单（US8）

### 8.1 取消流程
- [ ] 待支付订单详情页显示"取消订单"按钮
- [ ] 点击"取消订单"，显示确认弹窗
- [ ] 弹窗说明取消后果（如释放优惠券锁定）

### 8.2 取消成功
- [ ] 取消成功后，显示成功提示

### 8.3 取消失败

### 8.4 列表同步
