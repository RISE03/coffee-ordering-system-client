# 首页 UI 重构创意方案

> **设计目标**：弱化传统电商/点餐的功能性，极致强化品牌“朝暮”的**时间概念**，追求视觉上的精美与艺术感，打造毕业设计的核心视觉亮点。

---

## 方案 A：【光阴刻度】(The Chronos Dial)
**—— 抽象、现代、艺术装置感**

此方案将“时间”具象化为一个巨大的、不断流动的**“日夜轮盘”**，让用户直观地感觉到时间在页面上的流淌。

### 1. 布局结构示意

```text
  ┌─────────────────────────────────────────────────────┐
  │  Logo (Dawn & Dusk)                       Menu ☰    │
  │                                                     │
  │           "IT'S"              ▒▒▒▒▒▒▒▒▒▒▒           │
  │           TIME                ▒▒ 15:00 ▒▒           │
  │            FOR                ▒▒(轮盘区)▒▒           │
  │          COFFEE               ▒▒▒▒▒▒▒▒▒▒▒           │
  │                                     ▲               │
  │                                 (指针指示)           │
  │      (消隐呼吸文字)                                   │
  │                                 ┌───────┐           │
  │                                 │ 悬浮  │           │
  │     [ EXPLORE > ]               │ 商品  │           │
  │                                 └───────┘           │
  └─────────────────────────────────────────────────────┘
```

### 2. 核心视觉元素 (Hero Section)
*   **动态日晷/轮盘**：
    *   在屏幕中央（或右半部分）放置一个巨大的半透明圆环装置。
    *   圆环材质设定：一半是**金色的晨曦纹理**（象征 Dawn），一半是**深蓝的星空纹理**（象征 Dusk）。
*   **实时转动机制**：
    *   轮盘根据当前系统时间实时旋转。例如，下午 3 点时，指针精准指向“午·憩”区域。
    *   **光影追踪**：鼠标移动时，轮盘上会有微弱的聚光灯或光晕效果跟随，仿佛用户在手动探索时间。

### 3. 排版与布局
*   **左侧留白与排版**：
    *   使用巨大的衬线体（Serif Font）显示当前时段的问候语（如 *"It's Time for Coffee"* 或 *“此刻·午后”*）。
    *   文字采用“消隐”效果，随时间流逝缓慢呼吸。
*   **右侧视觉融合**：
    *   轮盘中心与当前推荐的一款“明星单品”重叠展示，形成“画中画”效果，商品仿佛悬浮在时间轴中心。

### 4. 交互体验
*   **手动拨动时间**：
    *   用户可以拖拽轮盘，模拟“拨动时间”。
    *   当轮盘转动跨越日夜界限时，全站背景色在“朝”与“暮”之间快速平滑切换（非常适合答辩演示）。

### 5. 风格定位
*   **关键词**：科技感、极简主义、高端奢侈品网页风格、概念艺术。

## 方案 B：【窗外·此刻】(The Window of Moment)
**—— 沉浸、情感化、电影感**

此方案将屏幕模拟成咖啡馆的一扇**“落地窗”**，强调用户“身在此时此刻”的情感体验。

### 1. 布局结构示意

```text
  ┌─────────────────────────────────────────────────────┐
  │  (全屏动态背景：窗外景色 + 玻璃雨滴/阳光特效)             │
  │                                                     │
  │   Morning Awake.              [ 10:00 AM ☀️ ]       │
  │   (超大背景文字)               (极简玻璃状态栏)        │
  │                                                     │
  │                                                     │
  │                                                     │
  │      ┌──────┐      ┌──────┐      ┌──────┐           │
  │      │精选 1│      │精选 2│      │精选 3│           │
  │      │玻璃卡│      │玻璃卡│      │玻璃卡│           │
  │      └──────┘      └──────┘      └──────┘           │
  │    (采用 Glassmorphism 风格，仿佛置于桌面上)           │
  │                                                     │
  │   ↓ Scroll to feel the time pass                    │
  └─────────────────────────────────────────────────────┘
```

### 2. 核心视觉元素 (Hero Section)
*   **全屏动态视窗**：
    *   **朝 (Dawn)**：画面是阳光透过树叶洒在木桌上的斑驳光影（丁达尔效应），配合微风吹动窗帘的轻微动态（CSS Animation）。
    *   **暮 (Dusk)**：画面是窗外模糊的城市霓虹与玻璃上的雨滴（Glass Rain Effect），室内是一盏温暖的复古台灯。
*   **玻璃拟态 (Glassmorphism)**：
    *   所有的文字、按钮和商品不再直接贴在背景上，而是仿佛写在布满雾气的玻璃窗上，或者悬浮的磨砂玻璃卡片，拥有真实的模糊与折射效果。

### 3. 排版与布局
*   **杂志封面感**：
    *   标题文字（如“朝·醒”）不仅是标题，而是作为背景纹理的一部分，使用超大字号、低透明度融入背景，像一本时尚杂志的封面。
*   **限制性展示 (Curated Display)**：
    *   **仅展示 3 个商品**：首页不再铺满商品，而是精选 3 款。这三款商品像艺术品一样“摆放”在桌面上，而非列表排列。

### 4. 交互体验
*   **日夜交替动画**：
    *   切换主题时，不仅仅是颜色变化，而是整个“窗外景色”的电影级淡入淡出（Cross-fade），伴随光影位置的移动。
*   **时间轴滚动**：
    *   页面向下滚动时，背景保持固定，但光影角度发生变化，仿佛用户坐在那里从早晨坐到了晚上。

### 5. 风格定位
*   **关键词**：日系小清新、韩系咖啡馆、文艺电影风格、沉浸式体验。

## 方案 C：【时间流动】(Time Flow)
**—— 动态、趣味性、品牌强关联**

核心理念：根据当前时间动态展示不同内容，强化“朝暮”品牌概念，将一天的时间轴可视化。

### 1. 布局结构示意

```text
  ┌─────────────────────────────────────────────────────┐
  │  动态时间展示区                                       │
  │  ☀️ 14:32  下午茶时光                                │
  │  "慵懒的午后，来一杯拿铁如何？"                        │
  ├─────────────────────────────────────────────────────┤
  │  时间轴导航                                          │
  │  ◯────●────◯────◯────◯                             │
  │  晨光   午后   暮色   深夜   全日                      │
  │ (06-10)(10-14)(14-18)(18-22)(全部)                   │
  ├─────────────────────────────────────────────────────┤
  │  当前时段精选（大图瀑布流）                            │
  │  ┌────────────┐  ┌────────┐  ┌────────┐            │
  │  │            │  │        │  │        │            │
  │  │  主推商品   │  │ 商品2  │  │ 商品3  │            │
  │  │  (大卡片)  │  │        │  │        │            │
  │  └────────────┘  └────────┘  └────────┘            │
  ├─────────────────────────────────────────────────────┤
  │  "日落倒计时：距离暮色模式还有 3小时42分"              │
  └─────────────────────────────────────────────────────┘
```

### 2. 详细模块说明

*   **动态时间展示区 (Hero Header)**：
    *   **实时状态**：显眼展示当前时间（如 `☀️ 14:32`）与时段名称（“下午茶时光”）。
    *   **动态文案**：配合时间的情感化文案，如 *"慵懒的午后，来一杯拿铁如何？"* 或 *"夜深了，用一杯热牛奶温暖梦境"*。

*   **时间轴导航 (Timeline Navigation)**：
    *   **可视化进度条**：横向展示一天的时间节点：晨光 (06-10) → 午后 (10-14) → 暮色 (14-18) → 深夜 (18-22)。
    *   **交互性**：用户点击不同时间节点，可预览该时段的推荐菜单与界面氛围。
    *   **当前指示**：高亮显示当前所处位置，并带有流动的进度动画。

*   **当前时段精选 (Curated Selection)**：
    *   **主推商品（Hero Card）**：占据左侧或中心大面积区域，高清大图展示本时段最推荐的一款单品。
    *   **辅助推荐**：右侧竖向排列 2-3 个次级推荐商品卡片。
    *   **视觉风格**：采用非对称的 Bento Grid（便当盒）或瀑布流布局。

*   **日落/日出倒计时 (Atmosphere Countdown)**：
    *   底部或悬浮展示，增加紧迫感与期待感。
    *   文案示例：*"日落倒计时：距离暮色模式还有 3小时42分"*。

### 3. 适合场景
*   **关键词**：功能与趣味平衡、品牌特色强化、游戏化体验、信息可视化。

## 推荐建议

考虑到现有素材（`dawn-bg.webp` / `dusk-bg.webp`）及技术栈（Vue + Tailwind）：

1.  **首选推荐：【方案 B：窗外·此刻】**。
    *   **理由**：利用 CSS `backdrop-filter` 和混合模式能以低成本实现电影级质感，最能体现“时间与空间”的沉浸感，视觉冲击力在毕设答辩中优势明显。

2.  **备选推荐：【方案 C：时间流动】**。
    *   **理由**：如果您希望在视觉精美的同时，保留更清晰的用户引导和趣味性交互（如时间轴切换），方案 C 是一个兼顾了功能性与品牌感的优秀选择。
