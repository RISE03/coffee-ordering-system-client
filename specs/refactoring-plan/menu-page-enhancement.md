# 点餐页面（MenuView）优化方案

> 创建日期：2026-02-02
> 状态：待评审

## 一、当前页面分析

### 优点
- 暮色主题配色统一，深蓝背景配金色调很有质感
- 左侧分类导航结构清晰
- 商品卡片采用玻璃拟态效果，符合品牌调性
- 顶部快捷操作按钮（购物车、结算、订单）便于访问

### 现存问题
- 商品卡片信息过于简单，仅显示名称和价格
- 缺少搜索和筛选功能
- 分类导航无选中状态高亮
- 缺少购物车快捷入口和数量提示
- 商品无法快速加购，需进入详情页

---

## 二、功能增强建议

### 2.1 搜索功能

**优先级：高**

| 功能点 | 说明 |
|--------|------|
| 搜索栏 | 页面顶部添加搜索输入框，支持按商品名称/关键词搜索 |
| 搜索历史 | 本地存储最近搜索记录，点击可快速搜索 |
| 热门搜索 | 展示热门搜索词，引导用户发现商品 |
| 搜索建议 | 输入时实时显示匹配的商品建议 |

### 2.2 筛选与排序

**优先级：中**

| 功能点 | 说明 |
|--------|------|
| 价格排序 | 支持按价格从低到高/从高到低排序 |
| 属性筛选 | 是否含咖啡因、冷/热饮、甜度等 |
| 标签筛选 | 新品、人气、限定、季节限定等 |
| 筛选器 UI | 顶部筛选条或侧边筛选面板 |

### 2.3 商品卡片增强

**优先级：高**

| 功能点 | 说明 |
|--------|------|
| 商品简介 | 卡片上显示 1-2 行商品描述文字 |
| 规格预览 | 显示可选规格标签（冰/热、大/中/小杯） |
| 销量/评分 | 增加社交证明，如"月售 500+"、"4.8 分" |
| 收藏按钮 | 卡片右上角添加收藏/取消收藏按钮 |
| 标签展示 | 新品、热卖、限定等标签醒目展示 |

### 2.4 快捷加购

**优先级：高**

| 功能点 | 说明 |
|--------|------|
| 加购按钮 | 商品卡片上直接显示"+"按钮 |
| 默认规格 | 点击"+"按钮以默认规格加入购物车 |
| 规格选择 | 点击卡片主体弹出规格选择弹窗 |
| 数量调整 | 已加购商品显示数量，支持"+/-"调整 |

### 2.5 购物车悬浮球

**优先级：中**

| 功能点 | 说明 |
|--------|------|
| 悬浮按钮 | 右下角固定显示购物车图标 |
| 数量角标 | 显示当前购物车商品总数 |
| 金额预览 | 显示购物车总金额 |
| 快速预览 | 点击展开购物车迷你列表 |
| 去结算 | 悬浮球内提供"去结算"快捷入口 |

### 2.6 商品收藏

**优先级：低**

| 功能点 | 说明 |
|--------|------|
| 收藏功能 | 用户可收藏喜欢的商品 |
| 收藏列表 | 个人中心查看收藏列表 |
| 收藏筛选 | 分类导航增加"我的收藏"入口 |

## 三、视觉美化建议

### 3.1 商品卡片优化

| 优化点 | 说明 |
|--------|------|
| 图片比例 | 统一所有商品图为 1:1 正方形比例 |
| 悬停动效 | 鼠标悬停时卡片上浮 4px，阴影加深 |
| 点击反馈 | 点击时卡片轻微缩放（scale: 0.98） |
| 标签样式 | 新品（绿色）、热卖（橙色）、限定（紫色）区分 |
| 图片加载 | 添加渐入动画，避免图片闪烁 |

```css
/* 悬停动效示例 */
.product-card {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}
.product-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.15);
}
```

### 3.2 分类导航美化

| 优化点 | 说明 |
|--------|------|
| 选中高亮 | 当前分类添加背景色或左侧指示条 |
| 分类图标 | 每个分类前添加小图标增强识别 |
| 数量角标 | 显示分类下商品数量，如"经典意式 (8)" |
| 悬停效果 | 鼠标悬停时背景色变化 |

```css
/* 选中状态示例 */
.category-item.active {
  background: rgba(var(--primary-rgb), 0.15);
  border-left: 3px solid var(--primary-color);
  color: var(--primary-color);
}
```

### 3.3 页面头部优化

| 优化点 | 说明 |
|--------|------|
| 分类 Banner | 切换分类时显示对应的氛围图/插画 |
| 商品计数 | 显示当前分类的商品数量 |
| 面包屑 | 显示当前位置：全部 > 经典意式 |

### 3.4 空状态设计

| 场景 | 设计 |
|------|------|
| 分类无商品 | 显示精美插画 + "暂无商品，敬请期待" |
| 搜索无结果 | 显示搜索插画 + "未找到相关商品" + 推荐热门 |
| 网络错误 | 显示错误插画 + 重试按钮 |

### 3.5 加载状态

| 场景 | 设计 |
|------|------|
| 页面加载 | 商品卡片骨架屏（Skeleton） |
| 图片加载 | 占位图 + 渐入动画 |
| 分类切换 | 淡入淡出过渡动画 |

```vue
<!-- 骨架屏示例 -->
<template>
  <div class="skeleton-card">
    <div class="skeleton-image"></div>
    <div class="skeleton-title"></div>
    <div class="skeleton-price"></div>
  </div>
</template>
```

### 3.6 响应式布局

| 屏幕尺寸 | 布局调整 |
|----------|----------|
| 桌面端 (>1024px) | 左侧分类 + 右侧 3-4 列商品网格 |
| 平板端 (768-1024px) | 顶部分类标签 + 2-3 列商品网格 |
| 移动端 (<768px) | 顶部横向滚动分类 + 2 列商品网格 |

## 四、交互体验优化

### 4.1 分类吸顶

- 滚动页面时，左侧分类导航固定在视口内
- 移动端分类标签栏吸顶

### 4.2 滚动联动

- 滚动商品区域时，左侧分类自动高亮对应分类
- 点击分类时，商品区域平滑滚动到对应位置

### 4.3 返回顶部

- 滚动超过一屏后，显示"返回顶部"按钮
- 点击平滑滚动回顶部

### 4.4 商品快速预览

- 点击商品弹出快速预览弹窗（Modal）
- 弹窗内可选择规格并加入购物车
- 无需跳转详情页，减少页面跳转

### 4.5 手势支持（移动端）

- 左右滑动切换分类
- 下拉刷新商品列表
- 长按商品卡片显示快捷操作
